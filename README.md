# Viligo Waste Warrior

An AI-inspired Food Waste Reduction and Pantry Management System built using React, TypeScript, and Vite.

## Features

* Pantry Inventory Management
* Expiry Tracking
* Food Waste Monitoring
* Shelf Life Calculator
* Dashboard Analytics
* Recipe Suggestions
* Sustainable Food Consumption

## Tech Stack

* React
* TypeScript
* Vite
* SQLite (Planned)
* Node.js
* Express

## Installation

```bash
npm install
npm run dev
```
